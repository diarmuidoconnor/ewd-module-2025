# Skills Repo

This repo contains reusable React, TypeScript, routing, API, and UI skills built up across the course.

## Purpose
The repo is used during spec-driven development to guide implementation.
Each skill should:
- explain the idea simply
- show when to use it
- include a small example
- highlight common mistakes
- be reusable across projects

## How to use it
1. Read the feature spec.
2. Identify the relevant skills.
3. Reuse existing patterns before inventing new ones.
4. Keep solutions understandable and close to course conventions.

## Design rule
Prefer small, explicit, type-safe patterns over clever abstractions.

## Current alignment
These skills match the course progression, including:
- props and iteration
- data down, action up
- routing
- reusable page templates
- component composition
- custom hooks
- typed API integration
- filtering
- caching of server state
